# nvm / nrm

  1. nvm 安装教程 (opens new window)

安装地址 (opens new window)

Nvm 路径 D:/nvm Node 路径 D:/Node

指令 功能
nvm install <version> 安装版本
nvm uninstall <version> 卸载版本
nvm use <version> 切换版本
nvm ls 查看版本列表
nvm install latest 安装最新版本

清除 npm 缓存方法： npm cache clean -f

  1. nrm 是一个可随时随地自由切换 Npm镜像 的管理工具，安装方式为： npm i -g nrm

nrm 指令如下

指令 功能
nrm add <name> <url> 新增镜像
nrm del <name> 删除镜像
nrm test <name> 测试镜像
nrm use <name> 切换镜像
nrm current 查看镜像
nrm ls 查看镜像列表
