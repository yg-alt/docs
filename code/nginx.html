# nginx 常见指令

指令 功能
nginx -s reopen 重启 Nginx
nginx -s reload 重新加载配置文件，优雅重启 推荐使用
nginx -s stop 强制停止
nginx -s quit 安全退出
nginx -t 检测配置文件地址 以及检测配置是否正常
nginx -v 显示版本信息并退出
killall nginx 杀死所有 nginx 进程
